function InternalSearchMonitor(const aHandle: PETEXT_DESC; var aComponent : TComponent): boolean;

Internal function used to find the TTesseractOCR component to trigger the TTesseractOCR.OnProgress event.

DON'T USE THIS FUNCTION.

function InternalRegisterMonitor(const aHandle: PETEXT_DESC; const aComponent : TComponent): boolean;

Internal function that adds a monitor and the parent TTesseractOCR component to a list.

This information is used to trigger the TTesseractOCR.OnProgress event.

DON'T USE THIS FUNCTION.

function InternalUnregisterMonitor(const aHandle: PETEXT_DESC): boolean;

Internal function that removes a monitor and the parent TTesseractOCR component from a list.

This information is used to trigger the TTesseractOCR.OnProgress event.

DON'T USE THIS FUNCTION.

property SetCurrentDir : boolean read FSetCurrentDir write FSetCurrentDir;
