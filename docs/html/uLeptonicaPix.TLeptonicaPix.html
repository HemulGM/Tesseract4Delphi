function Deskew(redsearch : integer = DEFAULT_REDUCTION_FACTOR) : TLeptonicaPix;

This binarizes if necessary and finds the skew angle. If the angle is large enough and there is sufficient confidence, it returns a deskewed image; otherwise, it returns a clone.

Typical values at 300 ppi for %redsearch are 2 and 4.

At 75 ppi, one should use %redsearch = 1.

<see href="https://github.com/DanBloomberg/leptonica/blob/master/src/skew.c">Leptonica source file: /src/skew.c (pixDeskew))
